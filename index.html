REACT

Qu'est-ce React ?



React est un framework connu sous le nom de React.js ou ReactJS. React est une bibliothèque JavaScript pour créer des interfaces utilisateur. Il est maintenu par Facebook et une communauté de développeurs et d’entreprises individuels. React peut être utilisé comme une base dans le développement d’une seule page des applications ou mobiles.

Avantages

  • Beaucoup plus facile à apprendre qu’Angular. React exige de bien revoir ses bases en HTML et ne réclame pas de syntaxe complexe.
  • React est centré sur JavaScript en encapsulant le HTML dans JS.
  • Flexibilité et responsive.
  • Une véritable boîte à outils offrant un très large choix aux développeurs.
  • 100% Open Source, beaucoup d’améliorations et de contributions régulières de la communauté des développeurs.
  • Réduction du nombre d’opérations sur le DOM (Document Object Model).
  • Optimisation et accélération du processus de mises à jour.

    • Inconvénients

  • Très peu de documentation officielle.
  • Pas de moteur pour injection de dépendances.
  • Malgré la légèreté de la syntaxe, Reactrequiert une connaissance approfondie pour intégrer l’interface utilisateur au framework MVC.

    • Actualités

    Svelte une alternative à React?
    La page Web moyenne est d’environ 2 mégaoctets, selon HTTP Archive , un site qui suit les performances des sites Web et les technologies qu’ils utilisent. Pour créer ces sites interactifs, de nombreux développeurs Web se tournent vers des packages open source qui gèrent les tâches courantes. Ces outils libèrent les programmeurs de beaucoup de travail de grognement, mais ils peuvent ajouter du poids à un projet. La bibliothèque React open source populaire de Facebook pour la construction d’interfaces utilisateur, par exemple, pèse 100 kilo-octets. Ajoutez quelques autres outils et graphiques, et bientôt vous parlerez de plusieurs mégaoctets.
    Le nouveau framework JavaScript Svelte , créé par le journaliste visuel et développeur de logiciels Rich Harris, vise à faciliter l’écriture de sites Web et d’applications interactifs plus petits et plus rapides. Le développeur Web Shawn Wang a déclaré avoir réduit la taille de son site Web personnel de 187 kilo-octets à 9 kilo-octets en passant de React à Svelte.
